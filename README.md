# Healthy Lifestyle Reward System (Good Fitness)

A Software Engineering project designed to encourage healthy lifestyle habits through nutrition tracking, goal adherence, and reward-based motivation.

## Overview

Good Fitness is a health-focused platform that helps users maintain healthy eating habits by tracking meals, calculating nutritional goals, and rewarding consistency through a competitive leaderboard system.

The system combines nutrition tracking, gamification, and AI-assisted meal analysis to improve user engagement and long-term commitment to healthy living.

## Problem

Many people struggle to maintain healthy habits due to:

- Poor dietary consistency
- Lack of motivation
- Inaccurate food tracking
- Difficulty monitoring progress
- Limited accountability

Good Fitness addresses these challenges through personalized nutrition goals, meal verification, and reward-based incentives.

## System Workflow

1. User creates an account.
2. User enters personal health information.
3. System calculates calorie and macronutrient targets.
4. User uploads meal photos.
5. AI analyzes meals and estimates nutrition data.
6. Points are awarded based on goal adherence.
7. Users compete on monthly leaderboards.
8. Top performers receive rewards.
